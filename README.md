- Flutter
- Riverpod
- Shared preferences
- Clean Architeture

## Setup
- Use https://github.com/rengo16/weather_app.git then cd weather_app
- Install dependancies with flutter pub get
- Add your openweather API key in core/constants.dart
- Flutter run
